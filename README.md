# Snake-Game
A classic Snake Game built with Python, featuring smooth controls, score tracking, food generation, and game-over detection. A fun project to learn game development fundamentals with Python.

A classic Snake Game built using Python. Control the snake, collect food to grow longer, and try to achieve the highest score without colliding with the walls or yourself.

Features :-

- Smooth snake movement

- Random food generation

- Real-time score tracking

- Snake grows after eating food

- Collision detection with walls and snake body

- Game Over screen

- Simple and beginner-friendly code structure

Controls :-

Key	     Action
↑	      Move Up

↓	      Move Down

←	      Move Left

→	      Move Right

Future Improvements :-
